## Контрольні запитання

1. Що таке Git?  
   Відповідь: Git — це розподілена система контролю версій.

2. Яка різниця між `git pull` та `git fetch`?  
   Відповідь: `git fetch` завантажує зміни, але не зливає їх, тоді як `git pull` завантажує та одразу зливає зміни.
