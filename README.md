\# Тема: Основи роботи з Git та GitHub



\## Опис

Ця лабораторна робота присвячена вивченню системи контролю версій Git та роботі з віддаленими репозиторіями на GitHub.
## Мета заняття

\- Навчитися створювати репозиторії на GitHub.

\- Оволодіти основами роботи з гілками в Git.

\- Зрозуміти процес створення та об’єднання гілок.
